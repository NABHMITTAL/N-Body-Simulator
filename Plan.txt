# Project TARS: N-Body Gravity Simulator

**Mission:** Build a complete N-body gravity simulator from scratch that accurately simulates gravitational interactions between celestial bodies while learning physics, mathematics, algorithms, and performance optimization.

---

# Rules

* Finish one phase completely before starting the next.
* Every phase must leave the project in a working state.
* Commit code after every completed phase.
* Understand every line of code—don't copy blindly.
* Prioritize correctness over performance initially.

---

# Tech Stack

Language: Python 3

Libraries:

* NumPy
* Matplotlib (initial visualization)
* Pygame (optional later)
* Numba (optimization)
* CuPy or PyTorch (GPU acceleration, final phase)

IDE:

* VS Code

Version Control:

* Git + GitHub
